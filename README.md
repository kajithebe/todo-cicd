# Todo CI/CD App

A simple Todo application with a full CI/CD pipeline.

## Deployed App

https://todo-cicd-443g.onrender.com

## Tech Stack

- Backend: Node.js + Express
- Frontend: React + Vite
- CI/CD: GitHub Actions
- Hosting: Render

## CI/CD Pipeline

This app uses GitHub Actions for CI/CD with automatic deployment to Render.
